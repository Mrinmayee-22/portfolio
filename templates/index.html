Healthcare AI · Featured

Diabetic Retinopathy Classification

A deep learning diagnostic system that classifies retinal fundus images across five severity stages of diabetic retinopathy, enabling early-stage detection at scale.

  • Designed and trained a custom CNN architecture on the APTOS/Kaggle retinal image dataset with multi-class severity grading
  • Applied advanced augmentation (CLAHE, zoom, flip, rotation) to address class imbalance and improve generalization
  • Achieved high classification accuracy with Quadratic Weighted Kappa as the primary evaluation metric
  • Implemented Grad-CAM visualization to produce clinically interpretable heatmaps over lesion regions
  • Designed for deployment as a screening aid in low-resource healthcare environments

MedTermQuest — Medical NLP System

An NLP-powered information extraction system that identifies, disambiguates, and retrieves medical terminology from clinical text using transformer-based models.

  • Implemented semantic search for querying medical concepts using vector embeddings and cosine similarity
  • Designed an information retrieval layer to map extracted terms to standardized medical ontologies (SNOMED/ICD-10)
  • Enables downstream clinical decision support and automated patient record analysis
